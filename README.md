# Contact-Book


# Overview
This project is a simple Contact Management System developed using Python. It allows users to store, view, search, update, and delete contact information. All contact data is stored in a JSON file, making the application fully offline and lightweight.

# Features
- Add new contacts (name, phone number, email)
- View all saved contacts
- Search for a contact by name
- Update existing contact details
- Delete contacts
- Automatic loading and saving of data using JSON

# Technologies Used
- Python 3.x
- JSON file handling
